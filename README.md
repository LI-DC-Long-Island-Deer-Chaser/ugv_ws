# LI-DC ROS2 Workspace
This is a repository containing all ros2 packages used in the LI-DC 2025-2026
Project. Each directory in `src\` is a git repository/submodule containing 
specific contents pretaining to each of the respective packages.

## Current Package List

- [ugv_description](https://github.com/LI-DC-Long-Island-Deer-Chaser/ugv_description)
- [Slamtec Lidar](https://github.com/Slamtec/sllidar_ros2.git)
